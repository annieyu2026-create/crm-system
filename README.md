# 客户管理系统 (CRM System)

一个功能完善、界面现代化的客户关系管理系统，包含用户认证、角色权限、客户管理、商机跟踪、交互记录和数据分析等核心功能。

## ✨ 核心功能

### 0. 用户认证与权限管理 ⭐ 新增
- 🔐 **用户注册与登录**：仅支持 `@fengfancloud.com` 企业邮箱
- 👥 **角色管理**：
  - **管理员**：查看所有数据，无查重限制
  - **普通用户**：仅查看自己数据，添加客户自动查重
- 🛡️ **客户查重**：自动检测邮箱和电话重复

### 1. 业务模块
- 📊 **数据概览**：实时统计与可视化图表
- 👥 **客户管理**：全字段记录，支持多人协作与产品标签
- 💼 **商机管理**：自动计算折扣与合同金额，跟踪交付状态
- 📞 **交互记录**：记录电话、会议、邮件等沟通细节
- 📈 **数据分析**：多维度图表分析

- HTML5 / CSS3 (TailwindCSS)
- JavaScript (ES6+)
